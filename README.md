## Sobre o Projeto

Desafio para a construção de uma interface fornecida pela Blue

## Como executar

```bash
# clonar o repo
git clone https://github.com/caio1902araujo/desafio-front-end.git

# entrar na pasta
cd desafio-front-end

# instalar as dependências
npm install

# executar o projeto
npm run dev
```
